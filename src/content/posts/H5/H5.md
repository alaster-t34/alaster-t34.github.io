# HTML簡介~~小貓娘~~
  HTML 指的是超文本編輯語言: HyperText Markup Language
  她不是編程語言，是標記語言，是一套markup tag
  HTML文檔（也被叫做web頁面）包括HTML標簽和文本内容。
# HTML標簽（元素）
  ##### <標簽>内容</標簽>
  ***HTML tag***
  這玩意是雙胞胎，*成對出現*。~~貓娘の耳朵~~
