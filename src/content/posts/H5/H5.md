# HTML5 小貓娘：基本資訊與使用方法

本頁面的存在目的，是協助使用者理解名為 **HTML5** 的 Web 結構技術。

為了降低理解成本，本頁將 HTML5 描述為一位負責搭建網頁骨架的 **小貓娘**。

她並非程式語言。

她是標記語言。

她的任務，是將文字、圖片、連結、表格、表單、影片與音訊整理成瀏覽器可以解析的結構。

換言之，HTML5 是 Web 頁面的骨架。

而 HTML5 小貓娘，就是負責把這副骨架排列整齊的執行者。

---

## 1. 基本定義

HTML 的全稱是：

```text
HyperText Markup Language
```

中文通常稱為：

```text
超文字標記語言
```

HTML5 則是現代 Web 常用的 HTML 標準。

它負責描述網頁內容的結構。

如果將一個網站看作一位角色，那麼：

| 技術         | 職責 | 小貓娘式說明   |
| ---------- | -- | -------- |
| HTML5      | 結構 | 骨架與房間配置  |
| CSS        | 樣式 | 衣服、髮型與顏色 |
| JavaScript | 行為 | 動作、互動與魔法 |

HTML5 小貓娘負責宣告：

> 這裡是標題。
> 這裡是段落。
> 這裡是圖片。
> 這裡是連結。
> 這裡是表單。
> 這裡是使用者可以操作的入口。

任務明確。

結構優先。

```text
將內容放到正確的位置。
```

這就是她作為 HTML5 的存在意義。

## 3. 第一個 HTML5 文件

一份最基本的 HTML5 文件如下：

```html
<!doctype html>
<html lang="zh-TW">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>HTML5 小貓娘</title>
  </head>
  <body>
    <h1>你好，我是 HTML5 小貓娘</h1>
    <p>我的任務是協助你建立清楚的網頁結構。</p>
  </body>
</html>
```

這段程式碼可以存成：

```text
index.html
```

然後用瀏覽器開啟。

如果頁面顯示出標題與段落，代表 HTML5 小貓娘已經成功啟動。

---

## 4. 文件結構解析

### `<!doctype html>`

```html
<!doctype html>
```

這一行告訴瀏覽器：

```text
本文件使用 HTML5 標準。
```

這是啟動宣告。

沒有它，瀏覽器可能會用較舊或不穩定的方式解析頁面。

---

### `<html>`

```html
<html lang="zh-TW">
</html>
```

`<html>` 是整份文件的根元素。

`lang="zh-TW"` 表示頁面主要語言是繁體中文。

這對搜尋引擎、翻譯工具與輔助閱讀器都有幫助。

---

### `<head>`

```html
<head>
  <meta charset="utf-8">
  <title>頁面標題</title>
</head>
```

`<head>` 裡放的是頁面設定。

這些內容通常不會直接顯示在畫面上。

常見用途包括：

* 設定文字編碼
* 設定頁面標題
* 設定手機版顯示方式
* 引入 CSS
* 設定 SEO 描述
* 設定網站 icon

---

### `<body>`

```html
<body>
  <h1>標題</h1>
  <p>段落</p>
</body>
```

`<body>` 裡放的是使用者真正看得到的內容。

HTML5 小貓娘會在這裡安排主要頁面結構。

## 5. 標籤 tag 是什麼？

HTML 使用標籤描述內容。

一個常見標籤長這樣：

```html
<p>這是一段文字</p>
```

其中：

| 部分       | 說明   |
| -------- | ---- |
| `<p>`    | 開始標籤 |
| `這是一段文字` | 內容   |
| `</p>`   | 結束標籤 |

可以理解為：

```text
HTML5 小貓娘打開一個盒子，放入內容，然後把盒子關上。
```

---

## 6. 常用標籤

### 標題

```html
<h1>主標題</h1>
<h2>副標題</h2>
<h3>小標題</h3>
```

標題用來建立文章層級。

一般建議一個頁面只使用一個 `<h1>`。

---

### 段落

```html
<p>這是一段正文內容。</p>
```

段落用來放主要文字。

如果只是普通文章內容，應該使用 `<p>`，不要用大量 `<br>` 強行換行。

---

### 連結

```html
<a href="https://example.com">前往 example</a>
```

`<a>` 是連結標籤。

`href` 是連結目標。

如果想在新分頁開啟，可以寫：

```html
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  開啟新分頁
</a>
```

這是一扇通往其他頁面的門。

---

### 圖片

```html
<img src="./h.jpg" alt="HTML5 小貓娘">
```

`src` 是圖片位置。

`alt` 是圖片說明。

如果圖片載入失敗，或使用者使用螢幕閱讀器，`alt` 會提供必要資訊。

因此，`alt` 不是裝飾。

它是輔助資訊。

---

### 列表

無序列表：

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
```

有序列表：

```html
<ol>
  <li>建立 HTML 文件</li>
  <li>加入 CSS 樣式</li>
  <li>加入 JavaScript 互動</li>
</ol>
```

如果內容有順序，使用 `<ol>`。

如果只是項目集合，使用 `<ul>`。

---

## 7. 屬性 attribute

屬性用來補充標籤設定。

例如：

```html
<a href="https://example.com" target="_blank">連結</a>
```

其中：

| 屬性       | 用途     |
| -------- | ------ |
| `href`   | 設定連結目標 |
| `target` | 設定開啟方式 |

屬性的格式是：

```text
屬性名稱="屬性值"
```

HTML5 小貓娘的判斷如下：

```text
標籤決定這是什麼。
屬性決定它如何運作。
```

## 8. HTML5 語義化標籤

HTML5 不只是把內容放到頁面上。

它還要求內容具有語義。

常見語義化標籤如下：

| 標籤          | 用途      |
| ----------- | ------- |
| `<header>`  | 頁首或區塊開頭 |
| `<nav>`     | 導覽列     |
| `<main>`    | 頁面主要內容  |
| `<article>` | 獨立文章    |
| `<section>` | 主題區塊    |
| `<aside>`   | 側邊資訊    |
| `<footer>`  | 頁尾      |

範例：

```html
<header>
  <h1>HTML5 小貓娘資料庫</h1>
</header>

<nav>
  <a href="/">首頁</a>
  <a href="/posts/">文章</a>
</nav>

<main>
  <article>
    <h2>HTML5 是什麼？</h2>
    <p>HTML5 是用來建立網頁結構的標記語言。</p>
  </article>
</main>

<footer>
  <p>任務完成。</p>
</footer>
```

這種寫法比全部使用 `<div>` 更清楚。

因為瀏覽器、搜尋引擎與輔助工具都能理解頁面結構。

---

## 9. 表格

表格適合呈現資料。

```html
<table>
  <thead>
    <tr>
      <th>標籤</th>
      <th>用途</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>&lt;p&gt;</td>
      <td>段落</td>
    </tr>
    <tr>
      <td>&lt;a&gt;</td>
      <td>連結</td>
    </tr>
  </tbody>
</table>
```

注意：

```text
表格用於資料，不應用於整站排版。
```

整站排版應交給 CSS。

---

## 10. 表單

表單用於收集使用者輸入。

```html
<form>
  <label for="name">名字</label>
  <input id="name" name="name" type="text">

  <label for="message">留言</label>
  <textarea id="message" name="message"></textarea>

  <button type="submit">送出</button>
</form>
```

表單中的關鍵元素：

| 元素           | 用途   |
| ------------ | ---- |
| `<form>`     | 表單容器 |
| `<label>`    | 輸入說明 |
| `<input>`    | 單行輸入 |
| `<textarea>` | 多行輸入 |
| `<button>`   | 按鈕   |

`label` 的 `for` 應該對應輸入框的 `id`。

這樣可用性更好，也更符合無障礙設計。

---

## 11. 多媒體

HTML5 支援原生音訊與影片。

### 影片

```html
<video controls width="640">
  <source src="demo.mp4" type="video/mp4">
  你的瀏覽器不支援 video 標籤。
</video>
```

### 音訊

```html
<audio controls>
  <source src="demo.mp3" type="audio/mpeg">
  你的瀏覽器不支援 audio 標籤。
</audio>
```

`controls` 會顯示播放、暫停、音量等控制器。

## 12. 如何使用 HTML5

### 步驟一：建立檔案

建立一個檔案：

```text
index.html
```

---

### 步驟二：寫入基本結構

```html
<!doctype html>
<html lang="zh-TW">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>我的頁面</title>
  </head>
  <body>
    <h1>我的第一個網頁</h1>
    <p>這是 HTML5 建立的內容。</p>
  </body>
</html>
```

---

### 步驟三：用瀏覽器開啟

你可以直接雙擊 `index.html`。

也可以用 VS Code 的 Live Server 插件開啟。

---

### 步驟四：加入 CSS

```html
<link rel="stylesheet" href="./style.css">
```

然後建立：

```text
style.css
```

寫入：

```css
body {
  font-family: system-ui, sans-serif;
  background: #f7fbff;
  color: #1f2937;
}
```

---

### 步驟五：加入 JavaScript

```html
<script src="./main.js"></script>
```

然後建立：

```text
main.js
```

寫入：

```js
console.log('HTML5 小貓娘已啟動。');
```

## 14. 常見錯誤

### 錯誤一：忘記結束標籤

錯誤：

```html
<p>這是一段文字
```

正確：

```html
<p>這是一段文字</p>
```

---

### 錯誤二：圖片沒有 `alt`

不建議：

```html
<img src="./h.jpg">
```

建議：

```html
<img src="./h.jpg" alt="HTML5 小貓娘">
```

---

### 錯誤三：標題層級混亂

不建議：

```html
<h1>主標題</h1>
<h4>突然出現的副標題</h4>
```

建議：

```html
<h1>主標題</h1>
<h2>副標題</h2>
<h3>更細的段落標題</h3>
```

---

### 錯誤四：用 HTML 負責樣式

HTML 負責結構。

CSS 負責樣式。

JavaScript 負責行為。

不要把所有職責都交給 HTML。

HTML5 小貓娘會因此陷入職責混亂。
